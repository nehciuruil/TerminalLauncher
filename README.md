# TerminalLauncher v1.0
**Windows 全局快捷键终端启动器 | 64 位轻量工具**

一款专为开发者/运维打造的 Windows 桌面效率工具，通过**全局快捷键一键呼出各类终端**，支持普通/管理员权限、窗口记忆、开机自启、热键自定义等实用功能，单文件运行、无广告、无后台臃肿。

---

## 🖥️ 运行环境
- 系统：Windows 10 / Windows 11（**64 位专用**）
- 架构：x64

---

## ✨ 核心功能
- 全局快捷键一键启动：CMD、PowerShell、Windows Terminal、Git Bash
- 支持**普通权限 + 管理员权限**双模式启动
- 自动记忆终端**窗口位置 & 大小**
- 系统托盘常驻，支持右键快捷菜单
- 热键冲突自动检测、程序防多开
- 支持**开机自启、终端置顶、默认终端切换**
- 支持配置文件 `config.ini` 自定义热键
- 快速在**桌面/文档/自定义目录**启动终端
- 一键复制当前资源管理器路径到剪贴板

---

## ⌨️ 默认全局快捷键
| 快捷键 | 功能 |
|--------|------|
| `Alt + C` | 启动普通 CMD |
| `Alt + A` | 启动**管理员** CMD |
| `Alt + P` | 启动普通 PowerShell |
| `Alt + D` | 启动**管理员** PowerShell |
| `Alt + W` | 启动普通 Windows Terminal |
| `Alt + S` | 启动**管理员** Windows Terminal |
| `Alt + G` | 启动普通 Git Bash |
| `Alt + H` | 启动**管理员** Git Bash |
| `Alt + V` | 复制当前资源管理器路径 |
| `Alt + X` | 退出程序 |

---

> 体积约 **150KB**，纯静态链接，无任何依赖。

## 📝 使用说明
1. 运行 `TerminalLauncher.exe`

---

## ⚙️ 自定义配置
程序首次运行自动生成 `config.ini`
可手动修改：
- 全局热键（支持 Alt / Ctrl / Win 组合）
- 终端默认窗口大小与位置
- 窗口置顶、开机自启等开关
